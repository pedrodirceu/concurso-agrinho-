# Agrinho 2026

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

---

## Objetivo do Projeto

O projeto tem como objetivo conscientizar sobre a importância do agronegócio sustentável, mostrando como a tecnologia pode aumentar a produção agrícola sem prejudicar o meio ambiente.

---

## Tecnologia Utilizadas

- HTML5
- CSS3
- JavaScript

---

## Funcionalidades

- Layout responsivo
- Tema claro/escuro
- Scroll suave
- Contador animado
- Formulário com validação
- Efeitos visuais interativos
- Acessibilidade por teclado

---

## Estrutura do Projeto

- `index.html` → Estrutura principal do site
- `style.css` → Estilização e responsividade
- `script.js` → Funcionalidades e interatividade

---

## Responsividade

O projeto utiliza Media Queries para adaptação em celulares, tablets e computadores.

---

## Acessibilidade

O site possui:
- navegação simples;
- contraste visual;
- suporte ao teclado;
- organização semântica.

---
